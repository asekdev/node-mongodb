# YE Says

## Description

YeSays is a CLI that returns quotes from them man himself, Kanye West.

## Installation

Clone or Fork the project, and then run the following to install.

Run:

```bash
npm install
```

Then:

```bash
npm link
```

## Usage

```bash
yesays quote
```

## Example Response

```bash
$ yesays quote
"Burn that excel spread sheet" -Ye
```
